Scientific Reports — Nature Portfolio (2026)

Federated MobileNetV2 with ensemble meta-learning for privacy-preserving brain tumor classification

Authors: Bhavesh Barmashe, Ankitesh Tiwari

This study introduces a privacy-preserving framework for identifying brain tumors from MRI scans without sharing sensitive patient data. It fine-tunes a MobileNetV2 model across multiple clients using FedAvg and FedProx to handle highly non-IID datasets. We train localized meta-learners (MLPs, SVMs, and Extreme Learning Machines) on client feature embeddings, achieving an elite 99.57% classification accuracy while significantly minimizing communication overhead.
